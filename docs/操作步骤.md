# 操作步骤

## 线上上传页面

```bash
cd /Users/hanchaoxu/Documents/GitHub/菁仕收费2
npm run dev
```

打开：

```text
http://localhost:4173
```

页面中选择或拖入最新 CSV 即可生成报表。CSV 只在浏览器本地处理，不会保存到服务器。

## 部署到 Vercel

```bash
cd /Users/hanchaoxu/Documents/GitHub/菁仕收费2
npx vercel --prod --yes
```

部署会使用：

```text
index.html
assets/app.js
assets/report-core.js
assets/styles.css
```

`.vercelignore` 会排除：

```text
data/raw/
outputs/
tests/
docs/
```

## 本地脚本生成

1. 将新的 CSV 保存为：

   ```text
   /Users/hanchaoxu/Documents/GitHub/菁仕收费2/data/raw/schedule.csv
   ```

2. 重新运行：

   ```bash
   cd /Users/hanchaoxu/Documents/GitHub/菁仕收费2
   python3 scripts/course_billing_report.py
   ```

3. 刷新或重新打开 HTML。

## 批量生成家长课时费明细

1. 将完整课时费 CSV 放入：

   ```text
   /Users/hanchaoxu/Documents/GitHub/菁仕收费2/data/完整课时费/
   ```

2. 确认原始课表为：

   ```text
   /Users/hanchaoxu/Documents/GitHub/菁仕收费2/data/raw/schedule.csv
   ```

3. 运行生成 HTML：

   ```bash
   cd /Users/hanchaoxu/Documents/GitHub/菁仕收费2
   npm run generate:student-billing -w @jingshi/parent-report
   ```

4. 查看输出：

   ```text
   /Users/hanchaoxu/Documents/GitHub/菁仕收费2/outputs/parent_reports/generated/
   ```

学生名会自动去掉末尾编号，例如 `Daniel 周恩东-3735` 输出为 `Daniel 周恩东-2026-03.html`。师资页会保留「授课老师」和「更多老师」。

如果要直接生成最终发给家长的连续长页 PDF，运行：

```bash
cd /Users/hanchaoxu/Documents/GitHub/菁仕收费2
npm run generate:student-billing-pdf -w @jingshi/parent-report
```

PDF 输出目录：

```text
/Users/hanchaoxu/Documents/GitHub/菁仕收费2/outputs/parent_reports/generated_pdfs/
```

PDF 是单页连续长页，文字为矢量输出，适合直接发给家长。
