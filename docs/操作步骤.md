# 操作步骤

## 线上上传页面

```bash
cd /Users/hanchaoxu/Documents/GitHub/菁仕收费2
npm run dev
```

打开：

```text
http://localhost:4173
```

页面中选择或拖入最新 CSV 即可生成报表。CSV 只在浏览器本地处理，不会保存到服务器。

## 部署到 Vercel

```bash
cd /Users/hanchaoxu/Documents/GitHub/菁仕收费2
npx vercel --prod --yes
```

部署会使用：

```text
index.html
assets/app.js
assets/report-core.js
assets/styles.css
```

`.vercelignore` 会排除：

```text
data/raw/
outputs/
tests/
docs/
```

## 本地脚本生成

1. 将新的 CSV 保存为：

   ```text
   /Users/hanchaoxu/Documents/GitHub/菁仕收费2/data/raw/schedule.csv
   ```

2. 重新运行：

   ```bash
   cd /Users/hanchaoxu/Documents/GitHub/菁仕收费2
   python3 scripts/course_billing_report.py
   ```

3. 刷新或重新打开 HTML。
